deployments/<username>/deployments.txt # This is a list of deployments by their names
deployments/<username>/<depname>.json # json for the details of the deployments

Deployment json structure
{
    "name": str # same as the filename
    "author": str # username of creator
    "git": str # Repository url
    "bash": str # stuff to run in the repo url after cloning or a restart
}



md_blog_content/currentid.txt # This is a file containing a single number to indicate the current latest post's id
md_blog_content/<id>.md # This is the post, it should contain a first line (Title), two newline charactera, third line containing the username of the author, two newlines again, and the content in freely formatted markdown.
